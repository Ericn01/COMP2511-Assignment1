Helping Chimpanzees All Around the Globe

The folks here at The Chimpanzee Foundation are constantly striving to improve our website's layout. By improving the design of our page, using modern web design features such as responsise design and grid layouts, we are also adding value to the user's experience. Easier web flow and better looking visuals that fit on any users screen, from smartphones to giant flat screen TV's is the name of the game, and we're all for it. Grid layouts allows a page to be structured into several columns and rows, each containing a "block" or container. This allows for a neatly with minimal effort, something that would've taken a lot of effort in the past.
